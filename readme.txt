A Wordpress Widget plugin to show data from custom post types.

== Description ==

By installing 'SP Custom Post Widget' plugin, you can easily show your data from custom post type in the widgets. 

SP Clients Carousel provides you with these options:

* Title - Let you specify custom title of the widgets box.
* Post Type - Lets you specify the post type of which you want to display datas in widget.
* No of items to show - For entering no of items to display in the widget.
* Show Excerpt - If you want to show post excerpt, then set it to yes.



== Installation ==

1. Upload the entire `sp-custom-post-widget` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu within the WordPress Dashboard.

You will now see the `SP Custom Widget` widget in your WordPress widgets section.
